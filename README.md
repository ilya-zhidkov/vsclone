# VSClone

A static template of popular Visual Studio Code editor.

---

## Setup

1. Clone or download repository.
2. Open `index.html` file using either local or file server.

---

### Supported Features

- [x] Light Theme
- [x] Dark Theme
- [x] Changing color scheme
- [x] Persisting current color setting
